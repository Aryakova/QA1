# Категории тестирования программного обеспечения

## 1. Функциональное тестирование - проверка соответствия ПО требованиям и спецификациям.

| №  | Вид тестирования       | Описание                                                                 |
|----|------------------------|--------------------------------------------------------------------------|
| 1  | Модульное (Unit)       | Тестирование отдельных компонентов кода (функций, методов)              |
| 2  | Интеграционное         | Проверка взаимодействия модулей/систем между собой                      |
| 3  | Системное              | Тестирование всей системы в среде с целью проверки соответствия системы исходным требованиям.                 |
| 4  | Регрессионное          | Повторная проверка после изменений кода                                 |
| 5  | Дымовое (Smoke)        | Быстрая проверка базовой функциональности                               |
| 6  | Санитарное (Sanity)    | Узкая проверка после исправления бага                                   |
| 7  | Приемочное (UAT)       | Тестирование заказчиком перед релизом                                   |

## 2. Нефункциональное тестирование - проверка характеристик производительности, безопасности и других аспектов.

| №  | Вид тестирования         | Описание                                                              |
|----|--------------------------|----------------------------------------------------------------------|
| 8  | Нагрузочное              | Оценка производительности под нагрузкой                              |
| 9  | Стрессовое               | Проверка работы в условиях превышения предельных нагрузок            |
| 10 | Тестирование стабильности| Проверка работы системы при длительной нагрузке                      |
| 11 | Тестирование безопасности| Поиск уязвимостей и проверка защиты от атак                         |
| 12 | Юзабилити                | Оценка удобства и понятности интерфейса                             |
| 13 | Тестирование доступности | Проверка соответствия стандартам доступности                        |
| 14 | Конфигурационное         | Проверка работы при разных настройках окружения                     |

## 3. Тестирование совместимости - проверка работы приложения в различных окружениях.

| №  | Вид тестирования         | Описание                                                              |
|----|--------------------------|----------------------------------------------------------------------|
| 15 | Кросс-браузерное         | Проверка в разных браузерах                                         |
| 16 | Кросс-платформенное      | Тестирование на разных операционных системах                         |
| 17 | Мобильное                | Проверка работы на мобильных устройствах                             |
| 18 | Локализация              | Тестирование адаптации под разные языки и региональные стандарты     |

## 4. Специальные виды тестирования

| №  | Вид тестирования         | Описание                                                              |
|----|--------------------------|----------------------------------------------------------------------|
| 19 | API-тестирование         | Проверка корректности работы API-интерфейсов                        |
| 20 | Тестирование баз данных  | Проверка целостности данных, SQL-запросов и миграций                |
| 21 | Хаотичное тестирование   | Имитация сбоев для проверки отказоустойчивости                      |
| 22 | Тестирование AI/ML       | Валидация алгоритмов машинного обучения и качества данных           |

## 5. Другие важные виды тестирования

| №  | Вид тестирования         | Описание                                                              |
|----|--------------------------|----------------------------------------------------------------------|
| 23 | Альфа-тестирование       | Внутреннее тестирование силами компании                              |
| 24 | Бета-тестирование        | Публичное тестирование реальными пользователями                      |
| 25 | Тестирование установки   | Проверка процесса инсталляции и деинсталляции                        |
| 26 | Тестирование отката      | Проверка восстановления системы после сбоя                           |
